~/projects Proyectos/Emprendimiento

Sistema de Gestión, Cotización e Inventario para Manufactura 3D/2D

Plataforma web diseñada para automatizar el ciclo de venta de un negocio de impresión 3D y corte 2D. Implementa un cotizador automático en tiempo real que analiza archivos .STL mediante Three.js (WebGL) y la API de Kiri:Moto desde el navegador; el backend desarrollado en PHP (MVC) asegura la consistencia de inventario mediante transacciones en PostgreSQL e integra pagos seguros con la API de PayPal.

Homelab Health Dashboard

Sistema de observabilidad en tiempo real para monitorear salud ambiental y consumo energético de servidores locales. Sensores de grado industrial conectados vía ESP32S3 transmiten por MQTT; un puente en Python almacena los datos en una base de datos de series de tiempo InfluxDB, expuestos en un dashboard de Grafana orquestado con Docker Compose y con alertas por Telegram bajo acceso seguro vía Tailscale.
